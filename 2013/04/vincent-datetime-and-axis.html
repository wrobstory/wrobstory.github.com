One of the things that not necessarily very intuitive with Vega is creating simple axis titles. Right now the only way I've figured out how to do it is to add custom data for your axis titles, and then mark instances referencing said data: 

    "data": [
    {
        "name": "table",
        "url": "data.json"
    },
    {
        "name": "y_label",
        "values": [
            {
                "label": "AAPL Price"
            }
        ]
    },
    {
        "name": "title",
        "values": [
            {
                "label": "AAPL Stock Price 1/1/2010-1/1/2013"
            }
        ]
    }
]
Proudly powered by Pelican, which takes great advantage of Python.
